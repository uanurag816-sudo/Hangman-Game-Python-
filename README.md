
# 🎮 Hangman Game (Python)

## 📌 Project Overview

This is a simple text-based **Hangman Game** developed in Python. The game randomly selects a word from a predefined list, and the player must guess the word one letter at a time. The player has a maximum of **6 incorrect guesses** before the game ends.

This project demonstrates the use of Python fundamentals such as loops, conditional statements, lists, strings, user input, and the `random` module.

## ✨ Features

* Randomly selects a word from a predefined list.
* Accepts one-letter guesses from the user.
* Displays correctly guessed letters and hides remaining letters with underscores.
* Prevents duplicate guesses.
* Validates user input.
* Allows up to 6 incorrect guesses.
* Displays a winning or losing message at the end of the game.
